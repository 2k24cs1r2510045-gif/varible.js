# JavaScript Function Scope and Variable Scope

## 📌 Project Overview

This project demonstrates the concept of **variable scope in JavaScript** using a simple HTML page. It explains how variables declared with `var`, `let`, and `const` behave in different scopes.

## ✨ Features

* Global variable demonstration
* Function scope using `var`
* Block scope using `let`
* Block scope using `const`
* Displays output in the browser
* Prints results in the browser console

## 🛠️ Technologies Used

* HTML5
* JavaScript (ES6)

## 📂 Project Structure

```
JavaScript-Scope/
│── index.html
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open the `create.html` file in any web browser.
3. Click the **Click Here** button.
4. View the output on the webpage and in the browser's Developer Console.

## 📖 Concepts Covered

* Global Scope
* Function Scope
* Block Scope
* `var`
* `let`
* `const`
* JavaScript Functions

## 🎯 Learning Outcomes

* Understand the difference between global, function, and block scope.
* Learn when to use `var`, `let`, and `const`.
* Practice creating and calling JavaScript functions.
* Improve understanding of variable accessibility in JavaScript.

## 📸 Output

When the button is clicked, the page displays:

* College Name

The same information is also logged in the browser console.
